# 📊 سیستم بک‌تست معاملاتی هوشمند (ارزهای دیجیتال)

یک سیستم بک‌تست دوطرفه (Long/Short) برای ارزهای دیجیتال با قابلیت انتخاب دستی نماد، تایم‌فریم و بازه زمانی.

## ✅ ویژگی‌ها
- پشتیبانی از صرافی‌هایی مانند KuCoin و Binance (via CCXT)
- سیگنال‌های ترکیبی (تکنیکال + منطق هوشمند)
- مدیریت ریسک با نسبت حداقل **2:1**
- گزارش دقیق: تعداد معاملات، حد سود/ضرر، نرخ برد
