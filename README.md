# 🚀 سیستم سیگنال BTC/USDT – 15 دقیقه‌ای – روی GitHub

این ربات به صورت خودکار هر 15 دقیقه BTC را تحلیل می‌کند و سیگنال ارسال می‌کند.

## ✅ ویژگی‌ها
- بدون نیاز به VPS یا نرم‌افزار جانبی
- فیلتر چندلایه حرفه‌ای (روندها، حجم، اصلاح، الگو)
- حد ضرر و سود مبتنی بر ساختار قیمت
- گزارش روزانه و پیام فعالیت هر ساعت

## 🔧 تنظیمات
1. توکن ربات تلگرام را در `Settings > Secrets` به نام `TELEGRAM_BOT_TOKEN` اضافه کنید.
2. (اختیاری) `TELEGRAM_CHAT_ID` را هم اضافه کنید.

## 📊 خروجی
- سیگنال‌ها در `signals.json` ذخیره می‌شوند.
- گزارش روزانه در UTC 00:00 ارسال می‌شود.
