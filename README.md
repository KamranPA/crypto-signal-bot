# 📊 سیستم بک‌تست کریپتو | KuCoin + تلگرام

یک سیستم بک‌تست ساده و قدرتمند برای شناسایی سیگنال‌های معاملاتی در تایم‌فریم ۱۵ دقیقه و ارسال آن به تلگرام.

## ✅ ویژگی‌ها
- دریافت داده از **KuCoin**
- تنظیم **ارز، تایم‌فریم، بازه زمانی** از طریق `config.yml`
- ارسال سیگنال به **ربات تلگرام**
- فقط بک‌تست (بدون معامله واقعی)
- مناسب برای **BTC, ETH, SOL** و سایر جفت‌ارزها
